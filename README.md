# Pokédex

Uma Pokédex feita com **HTML, CSS e JavaScript**, que permite buscar informações de Pokémons de forma interativa.  

## 🚀 Tecnologias
- HTML5
- CSS3
- JavaScript
- API: [PokéAPI](https://pokeapi.co/)
